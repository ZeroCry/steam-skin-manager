Steam Skin Manager's Help - Custom Skins

Installing a Custom skin

To install a custom skin, you must first extract it. It is highly reommended to put it into a place where it won't be bothering you so that you can leave it there.

This won't copy the folder, only link it with Steam. If you delete any custom theme and won't back it up, you'll loose it.

If you decide to move your skin folder elsewhere, you'll have to set it here again, which will overwrite previous settings.

Using 3rd party skins

Most skins available on the Internet support Microsoft Windows, some Mac OS X. Their appearance may be wrong on Linux. To use them you may have to tweak them a little bit.
There is no universal way to accomplish this since every skin may suffer from a different bug.

Please note, some skins work flawlessly on Linux, like the Windows 8 Metro skin
