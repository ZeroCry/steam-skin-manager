Steam Skin Manager's Help - Application Details

First Run

Steam Skin Manager helps you apply and use custom themes with Steam client. To set a theme, you must be at the Appearance tab. You can either use one of the bundled themes or you may want to use a different theme, either your own or a downloaded one.

Using both bundled or custom themes is simple. You only need to select one of the listed Bundled (preinstalled) Skins and hit apply. If you wish to use native window decorations, then switch on the most upper button.

If you select a custom skin using the file dialog chooser, Steam Skin Manager will automatically expect you wish to apply a 3rd party skin and will act accordingly. In this case you must have your theme extracted and placed somewhere you wish it to stay. Then select the related skin folder through the dialog and hit apply. Read more »
