Editing registry.vdf file manually

registry.vdf file contains Steam client settings. Some can only be changed here.

Basics

"HKCU"
 {
  "Software"
   {
    "Valve"
      {
       "Steam"
         {
          "SkinV4"	"MY_SKIN"
          "Language"	"MY_LANGUAGE"

e.g. to change interface language, replace "MY_LANGUAGE" with something like "english".
You can also change your skin here, but you must first put it into ~./steam/Steam/skins/MY_SKIN directory.
