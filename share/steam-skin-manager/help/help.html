Welcome to Steam Skin Manager

This tool allows you to install bundled and custom skins for Steam as well as revert to the original skin.

I want to install:
  1. a bundled skin
  2. my or downloaded skin
or I want to edit settings manually.

If you want to revert to the original skin, use menu item: Steam → Revert to the default skin.
